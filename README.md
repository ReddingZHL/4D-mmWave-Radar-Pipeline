# 4D毫米波雷达感知与多目标跟踪系统
**从FMCW回波仿真到多目标卡尔曼跟踪的完整Pipeline**

[![MATLAB](https://img.shields.io/badge/MATLAB-R202x-orange.svg)](https://mathworks.com) [![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## 1. 项目简介 
本项目完整实现了 **77/79GHz FMCW 4D毫米波雷达** 的信号处理与感知全链路，从原始回波仿真到多目标跟踪闭环，支持**点目标**和**扩展目标（飞机）**场景仿真。

**核心目标**：为毫米波雷达算法研究、车载ADAS、安检系统等提供可复用、高精度的仿真验证平台。



## 2. 项目亮点

- **完整信号处理链路**：FMCW回波生成 → Range-Doppler 2D-FFT → CA-CFAR检测 → TDM-MIMO相位补偿 → 1D-MUSIC超分辨角度估计 

- **4D点云生成**：支持笛卡尔坐标 (x, y, z, v) 转换 
- **多目标聚类**：DBSCAN、K-Means、X-Means（BIC自适应确定簇数）
- **多目标跟踪**：基于CV模型的卡尔曼滤波 + GNN关联 + 航迹生命周期管理（孵化/销毁） 
- **高精度指标**（50帧仿真）：  
  - 角度估计精度：±1°  
  - 径向速度RMSE：0.0343 m/s  
  - 聚类准确率：> 90%  
  - 航迹存活率：100%  
  - OSPA距离：0.8357 米

## 3. 项目结构

```
4D-mmWave-Radar-Pipeline/
├── generateParameter.m % 雷达及目标参数定义
├── echo_generation.m % 根据设定的雷达参数及目标参数，生成回波信号并存储在data文件夹中
├── generateSignal.m % 仿真FMCW点目标回波信号生成
├── all_frames_truth_plot.m % 仿真目标质心随帧运动真实轨迹绘制
├── main_pipeline.m % 主入口，读取回波文件并进行处理
├── signal_processing/ % 信号处理模块
│ ├── range_doppler_fft.m
│ ├── cfar_detection_1D.m、cfar_detection_1D_pro.m、cfar_detection_2D.m % 分别为1D级联for循环cfar检测、1D级联快速矩阵cfar检测、2Dcfar检测
│ ├── doa.m % cfar检测点的角度估计
│ └── pointcloud_generation.m % 点云生成
├── post_processing/ % 后处理模块
│ ├── points_clustering.m % 点云聚类
│ ├── extract_centroids.m % 提炼点云聚类质心
│ ├── calculate_radar_metrics.m % 计算每帧的检出率，虚警率，平均角度估计误差
│ ├── kalman_tracking.m % 根据多帧得到的聚类质心，卡尔曼滤波轨迹预测
│ └── plot_tracking_results.m % 绘制（多）目标跟踪轨迹及航速航向误差演进图并保存在results文件夹内
├── results/ % 运行结果图
├── data/ % 仿真回波数据
└── README.md
```

## 5. 运行 (Quick Start)

1. 克隆本项目到本地：

   Bash

   ```
   git clone [https://github.com/你的用户名/你的项目名.git](https://github.com/你的用户名/你的项目名.git)
   cd 你的项目名
   ```

2. 打开 MATLAB 。

3. 先打开并运行echo_generation.m，生成回波信号，可在echo_generation.m中设置targetnum，1表示生成点目标，其他数字表示生成飞机目标。运行完成后会在data文件夹下生成一个radar_pipeline_inputs.mat文件，该文件存储了仿真回波数据，目标真实质心轨迹及SNR，雷达参数指标。

4. 打开并运行main_pipeline.m，可获得下节描述的运行效果。
