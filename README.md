## 🌟 项目简介 (Project Overview)
本项目全线打通了汽车/防空毫米波雷达感知的完整数据流链条。系统包含两大核心版本：
* **Version 1 (纯仿真沙箱)**：支持自定义多目标（中距离斜切、远距离左侧斜穿、近距离大角度脱离），包含雷达时频回波生成、加窗 2D-FFT（距离-多普勒谱）解算、CFAR 恒虚警检测、DBSCAN 聚类。
* **Version 2 (真实数据集扩展)**：提供面向公开数据集（如 **View-of-Delft (VoD)**、**TJ4DRadSet**）的通用 4D 雷达数据适配器接口，用于实车真实工况的工程鲁棒性验证。

后级跟踪器采用全维状态联动的**卡尔曼滤波器 (Kalman Filter)**，配合 **GNN (全局最近邻)** 关联矩阵与动态生命周期管理。核心攻克了雷达多普勒一维盲区和矢量投影符号错乱的历史难题，实现了低于 **1%** 的工业级稳态航速与航向角估计精度。

## 🚀 核心技术亮点 (Technical Highlights)

1. **一维多普勒速度矢量自洽对账**

   通过直角坐标系下的位置矢量与速度状态向量的**动态点积 (Dot Product) 判定法**：

   $$\text{航速符号} = \text{sign}(X \cdot v_x + Y \cdot v_y)$$

   完美解决了雷达“靠近为负、远离为正”的符号映射，确保滤波器暂态与稳态估计值与上帝视角真值严丝合缝。

2. **后 10 帧滑动窗口平滑算法 (Moving Window Average)**

   剔除了卡尔曼建立航迹初期的“孵化暂态坏账”，在稳态期利用最优估计进行动态平滑，成功将全场航速相对误差收敛至 **$< 1\%$**。

3. **大一统多目标评估指标 (OSPA Distance)**

   引入 OSPA（最优子模式分配）阶数 $p=2$, 惩罚 $c=10$ 作为终局考核，平均 OSPA 达到 **`0.8998 米`**，强力证明系统无虚警、无漏检、无航迹掉包（Identity Switch）。

## 🏁 如何一键运行 (Quick Start)

1. 克隆本项目到本地：

   Bash

   ```
   git clone [https://github.com/你的用户名/你的项目名.git](https://github.com/你的用户名/你的项目名.git)

   ```
